RGB Setting Way

The arttribute order of the RGB method is Red, Green then Blue

And its setting value is between 0 to 255

Red Font: RGB(255,0,0)

Green Font: RGB(0,255,0)

Blue Font: RGB(0,0,255)

White Font: RBG(255,255,255)

Black Font: RGB(0,0,0)

RGBA

A is an abbreviation for Alpha, that means opacity.

Setting value of opacity is between 0 to 1

HEX Setting Way

The arttribue order of Hexadecimal method is Red, Green then Blue also.

Unlike RGB, the setting value is between 00 to ff.

Red Font: #ff0000

Green Font: #00ff00

Blue Font: #0000ff

White Font: #ffffff

Black Font: #000000

Hexadecimal doesn't have opacity like RGB.

HSL

HSL is abbreviation for Hue, Saturation and Lightness.

Hue

Hue is a degree of color circle. You can set it between 0 to 360.The order is Red, Green then Blue also.

When Hue is 0, the color is Red.

When 120, the color is Green.

When 240, the color is Blue.

When 360, the color is back to Red.

Saturation

Its value is between 0% to 100%,representing gray to pure color.

This is Red.But saturation is 0%

This is Red.But saturation is 50%

This is Red also.But saturation is 100%

Lightness

Its value is between 0% to 100%,representing no light(black) to full light(white)

This is Green,but Lightness is 0%

This is Green,but Lightness is 50%

This is Green,but Lightness is 100%

HSLA

As same as RGBA, opacity can be set.

Value is between 0 to 1.
