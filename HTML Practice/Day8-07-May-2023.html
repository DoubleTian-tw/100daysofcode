Image Map

what is map?

Map will define an area which clickable area.

Usemap attribute

Usemap is define in the img tag

<img usemap="#NAME">

Map tag

Map tag defined a clickable area.

<map name="NAME"> </map>

Area tag

Area tag locate the coordinates of items in the image.

<area shape="rect" coords="175,30,320,110" href="#" alt="Jacket" title="Jacket">

The meaning of each attribute :

  • SHAPE: This define a clickable area with rect(rectangular),circle,poly(polygonal) and default.
  • COORDS: This define the coordinates position of each shapes.

A clickable area can also trigger javasrcipt function.
