What is RWD?

RWD is a design pattern that make your website prettier on different device.

How RWD works?

Responsive Image

This picture setting is width=100%

This picture setting is max-width=100%

Showing different pictures depending on device-width.

When the width is less than 400px, you will see a gif.

When the width is between 400px and 800px, you will see a dom.

Beside those width, you will see a future view.

Responsive text

Responsive text can be set by "vw" which is called view width.

The text will resize by the device width.
