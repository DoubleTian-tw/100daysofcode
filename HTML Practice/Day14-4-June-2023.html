Head Element

<meta>

Here are some list of meta

  1. Character set
    • <mata charset="UTF-8">
  2. Define keyword for search engines
    • <meta name="keywords" content="100daysofcode">
    • - Google don't use anymore
    • - You can use description instead keywords.
  3. Define description for your web
    • <meta name="description" content="100daysofcode">
  4. Define author for your web.
    • <meta name="author" content="Tian">
  5. Refresh your website for specify times.
    • <meta http-equiv="refresh" content="30">
    • - content represent times(second)
  6. Fit your website on different device.
    • You can switch between different types of devices on F12.
