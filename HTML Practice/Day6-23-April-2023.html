CSS

Inline style

When setting a tag style, you could use tag attribute.

Like This Way. Style="color:YOUR_COLOR"

Internal style

The internal setting is located in head tag.

<head>
<style>
h2 {color: RGB(0,120,240);}
</style>
</head>

External style

The External style has similarities with internal.

Also set in the head tag. But there is a separate file called css.

And the tag is <link>

The font-family style used by p tag is the external style.
