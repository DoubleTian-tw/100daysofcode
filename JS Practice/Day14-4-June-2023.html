Javascript

Where can <script> be placed?

You can place it in <head> and <body>



How is exteral file works?

You can use <script> with src attribute.

Like <script src="YOUR_FILE.js">

It have some advantages on external file.

  • Separates with html and css.
  • Easy to read and maintain.
  • Speed up loads pages.

How to display data?

  • innerHTML - write into the html element.
  • document.write() - write into the html output.
  • window.alert() - write into the alert box, which will pop up.
  • console.log() - write into the console for test, it won't display on page.

Using document.write() after the page is loading, whole page will be delete.
